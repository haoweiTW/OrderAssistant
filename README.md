# OrderAssistant
MetaTrader4 GUI EA

PanelTool Button Click trading (Replace Default One-Click-Trading)

Features
01. Button Click open or close Long positions
02. Button Click open or close Short positions
03. Set Lot sizes
04. Set SL or TP points
05. Set order's commentment
06. Show BreakEven price
07. Show Long or Short Lots
08. Show Profit or Loss USD
09. Show Daily Profit or Loss USD
10. Set ID=0 to handle manual orders together
